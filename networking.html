DNS

The Domain Name System (DNS) translates domain names into IP addresses. When a user visits a website, DNS resolves the domain name into the IP address of the server hosting the site.

IP Addressing

Every device connected to the internet uses an IP address. Websites may resolve to both IPv4 and IPv6 addresses depending on the network configuration.

HTTP vs HTTPS

HTTP is used for communication between web clients and servers. HTTPS adds encryption using TLS, ensuring secure data transmission over the internet.
