# Networking Information Website

This repository contains a personal project website created by **Paul Lichtenwalner**, designed to provide educational content on networking concepts including DNS, IP addressing, and web protocols. The website also includes a contact page with links to professional profiles.

---

## Overview

This project is a simple static website that explains key networking concepts in a clear and accessible way. The website is built with **HTML and CSS** and hosted on **GitHub Pages**, making it publicly available. It is intended for educational purposes, showcasing a combination of technical knowledge and web development skills.

The website includes:

- Detailed explanations of DNS, IP addressing, and HTTP/HTTPS protocols.  
- Examples of DNS lookups with `nslookup`.  
- A network flow diagram demonstrating how a user connects to a website.  
- A professional contact page with an "About Me" section and clickable LinkedIn/GitHub icons.

---

## Features

- **Responsive Layout:** Uses CSS to create a clean, centered layout that works across devices.  
- **Educational Content:** Covers foundational networking topics with examples.  
- **Interactive Elements:** Clickable LinkedIn and GitHub icons on the contact page.  
- **Network Diagram:** Visual representation of how DNS and HTTPS work together.  
- **Hosted on GitHub Pages:** Accessible publicly via GitHub Pages hosting.

---

## Technologies Used

- **HTML5** – Structure and content of the website.  
- **CSS3** – Styling, layout, and responsive design.  
- **GitHub Pages** – Hosting and deployment of the static website.  
- **Git** – Version control.

---

## Setup and Usage

You can view the website directly on GitHub Pages or clone the repository to your local machine for development or editing.
