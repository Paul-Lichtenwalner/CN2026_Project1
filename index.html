Computer Networking Concepts

Public Website Deployment Project

Welcome

This website was created for a Computer Networking Project. The goal of this project is to deploy a publicly accessible website and demonstrate networking concepts such as DNS, IP addressing, and secure web protocols.

The website is hosted online and accessible through a public domain using HTTPS. Networking tools such as nslookup and browser developer tools are used to demonstrate how domain names resolve to IP addresses and how HTTP requests are transmitted securely.
